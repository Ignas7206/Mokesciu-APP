# Google Play Release Checklist

## Account

- [ ] Google Play developer account sukurtas.
- [ ] Tapatybe patvirtinta.
- [ ] Android device verification atliktas, jei Google paprase.

## App setup

- [ ] Create app.
- [ ] App name: `Mokesciu atsidejimas`.
- [ ] Default language: Lithuanian, jei yra; kitaip English ir veliau LT listing.
- [ ] App type: App.
- [ ] Pricing: Free.
- [ ] Category: Finance.
- [ ] Contact email: `ltdigitaltools@gmail.com`.
- [ ] Privacy policy URL: `https://ignas7206.github.io/Mokesciu-APP/privacy-policy.html`.

## Store listing

- [ ] Short description is `docs/play-store/store-listing-lt.md`.
- [ ] Full description is `docs/play-store/store-listing-lt.md`.
- [ ] App icon paruosiamas 512x512 PNG.
- [ ] Feature graphic paruosiamas 1024x500 PNG.
- [ ] Bent 2 telefono screenshots.

## Policy forms

- [ ] App access: all functionality available without special login.
- [ ] Ads: No ads.
- [ ] Content rating: use `docs/play-store/content-rating.md`.
- [ ] Target audience: adults / not designed for children.
- [ ] News apps: No.
- [ ] Financial features: tax estimation / personal finance tracking only, no loans, no banking, no investing.
- [ ] Data safety: use `docs/play-store/data-safety.md`.

## Testing

- [ ] Internal testing release uploaded.
- [ ] App works on your Android phone.
- [ ] Closed testing track created.
- [ ] 12 testers added.
- [ ] Testers opted in for 14 continuous days.
- [ ] Production access requested.

## Before public launch

- [ ] Matematikos nustatymai patikslinti.
- [ ] 30% atskaitymu ir realiu islaidu rezimai suprojektuoti.
- [ ] Pensiju kaupimo nustatymas pridetas arba aiskiai pazymeta, kad dar nera.
- [ ] PVM moketojo rezimas nepristatomas, kol nera padarytas.
- [ ] Privacy policy atnaujinta, jei atsiranda cloud sync, ads, analytics arba login.
